# SOT 索引（Single Source of Truth）

Last Updated: 2026-02-22

当前生效事实只看以下文件（其余计划性文档请放在 `wip/`）：

- `overview.md`：项目定义、repo 列表、最小本地开发路径
- `architecture.md`：模块边界、关键约束、跨模块交互

更新规则：
- 每次需求验收并确认后，才更新相关 SOT 文件，使其反映最新实现
- 需求 plan 属于过程信息，完成后放到 `docs/archive/YYYYMMDD-<topic>/`，避免与 SOT 混淆
- 提交前仅允许新增/修改本目录内的 SOT 条目，不修改 `archive` 历史记录
